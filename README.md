# Spotify Display (ESP32 + 16x2 LCD)

## what is this  
this is a small desk display that shows my currently playing spotify track using an esp32 and a 16x2 lcd screen (yeah i know… kinda basic but it gets the job done)

## why i made this  
i wanted to make something to start doing hardwar (ive done hardware before, but never actually done something from scratch)
